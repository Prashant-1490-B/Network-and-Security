# Enterprise Security Lab

This repository contains hands-on documentation of an enterprise-style
cybersecurity lab focused on internal networking, packet analysis, and
firewall-based attack surface control.

## Covered Phases
- Phase 1.1: Lab Architecture & Network Design
- Phase 1.2: Network Visibility and Packet Analysis

## Tools Used
- Wireshark
- UFW (iptables)
- Linux (Kali, Ubuntu)
